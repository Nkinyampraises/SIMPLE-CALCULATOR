# Contributing to Simple Interest Calculator

First off, thank you for considering contributing to this project! It’s people like you that make the open-source community such a great place to learn, inspire, and create.

## Welcome!
All contributions, bug reports, bug fixes, documentation improvements, enhancements, and ideas are welcome.

## How Can I Contribute?

### Reporting Bugs
* Check the Issues tab to see if the bug has already been reported.
* If not, open a new issue. Clearly describe the problem and include steps to reproduce the error.

### Suggesting Enhancements
* We are always looking for ways to make the calculator more "interesting" or efficient! 
* Please open an issue with the tag "enhancement" to discuss your idea.

### Pull Requests
1. Fork the repository.
2. Create your feature branch (`git checkout -b feature/AmazingFeature`).
3. Commit your changes (`git commit -m 'Add some AmazingFeature'`).
4. Push to the branch (`git push origin feature/AmazingFeature`).
5. Open a Pull Request.

## Documentation Improvements
Notice a typo in the README? Want to improve the math explanations? Documentation contributions are just as valuable as code!

**Thank you for your support!**
